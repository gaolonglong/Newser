# Newser
Newser是一个集知乎日报、微信精选、豆瓣一刻的聚合性的Android阅读客户端，提供精选内容的阅读、本地收藏、分享、简单换肤、运行时权限、夜间模式和离线缓存等功能。
增长姿势水平。
